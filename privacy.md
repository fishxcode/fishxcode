# FishXCode 隐私政策

**生效日期：2026年2月11日**

FishXCode 严格遵守《中华人民共和国个人信息保护法》、《网络安全法》、《数据安全法》及相关法律法规。

## 一、我们收集哪些信息及用途

| 场景 | 收集内容 | 用途 |
| --- | --- | --- |
| 账户注册 | 手机号或电子邮箱、密码、用户名 | 创建账号、验证身份、维护账户安全 |
| API 使用 | 调用时间、请求路径、消耗 Token 数 | 服务计费、防欺诈和滥用 |
| 设备与网络 | IP 地址、设备标识符、浏览器类型、操作系统 | 保障服务安全稳定运行 |
| 支付 | 由第三方（支付宝、微信支付）处理 | 完成交易；FishXCode 不存储完整银行卡信息 |
| 客服沟通 | 联系方式、通信记录 | 验证身份、解决问题 |

::: tip 关于模型训练
除非您主动开启"历史记录"或"调试模式"等可选功能，FishXCode **不会**使用您的输入或输出来训练 AI 模型。
:::

## 二、信息共享与披露

- **第三方模型提供商**：为响应您的 API 请求，我们必须将输入内容传输给您选择的提供商（建议您查阅其隐私政策）。
- **授权合作伙伴**：云服务商、数据分析服务商等，须签署严格的数据保护协议。
- **法律要求**：根据法律法规或政府主管部门要求，我们可能共享信息。
- 我们**不会出售**您的个人信息，不会进行未经授权的公开披露。

## 三、数据的跨境传输

FishXCode 运营实体位于中国境内，个人信息存储在境内。如确需跨境传输，将严格遵守国家网信部门规定，并依法取得您的单独同意。

## 四、存储与安全

- **存储期限**：仅在实现目的所必需期间及法律要求期限内保留；超期后删除或匿名化处理。
- **安全措施**：加密、去标识化、访问控制、定期安全审计。

## 五、您的权利

根据《个人信息保护法》，您享有以下权利：

- **查阅与复制**：查阅、复制您的个人信息
- **更正**：更正不准确或不完整的信息
- **删除**：在特定情形下申请删除
- **撤回同意**：随时撤回基于同意处理的授权
- **注销账户**：注销后将停止处理并依法删除或匿名化个人信息

行使权利请通过下方联系方式提交申请，我们将在 15 个工作日内答复（可能需要身份验证）。

## 六、未成年人保护

我们的服务主要面向成年人，**不会故意收集未成年人的个人信息**。如发现未成年人未经监护人同意提供了个人信息，请立即联系我们。

## 七、政策更新

若变更实质性影响您的权利，我们将通过网站显著位置公告或向注册邮箱发送通知告知您。
